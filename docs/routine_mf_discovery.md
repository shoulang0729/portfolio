# MF 口座棚卸し（初回のみ・Claude for Chrome）

目的: MoneyForward の全口座を一覧化し、`data/mf-import-config.json` の `exclude.accounts` に何を入れるかを決めるための「棚卸し」。**コミットはしない。出力を Toshio に見せるだけ。**

実行条件: Chrome で MoneyForward にログイン済みであること。

---

## プロンプト（Claude for Chrome にそのまま貼る）

あなたは私の資産棚卸しを手伝うアシスタントです。**ページの閲覧と読み取りのみ**を行い、いかなる編集・送信・発注・設定変更もしないでください。

手順:

1. `https://moneyforward.com/bs/portfolio`（資産内訳ページ）を開く。
2. ページ内の資産を**「金融機関/口座」単位**で洗い出す。各カテゴリ（株式(現物)・投資信託・債券・年金・暗号資産・預金/現金・FX/商品先物・その他）を展開し、どの口座にどのカテゴリの資産がいくらあるかを読む。
3. 次の Markdown テーブルだけを出力する（説明文は不要）:

| 金融機関/口座名 | 含まれるカテゴリ | 銘柄数 | 評価額合計(円) |
|---|---|---|---|
| （例）SBI証券 | 株式(現物), 投資信託 | 12 | 123,456,789 |
| … | … | … | … |

4. テーブルの最後に1行、`MF総資産（画面表示）: ◯◯◯円` を付ける。
5. 桁・カンマはそのまま。読めない口座があれば「読み取り不可」と明記。

出力はこれだけ。コミットや編集は絶対にしない。

---

## 出力を受け取ったあと（Toshio → MulmoClaude）

この口座一覧を MulmoClaude に貼れば、除外したい口座を選んで `mf-import-config.json` の `exclude.accounts` に反映する。これで定常運用（`routine_mf_snapshot.md`）の準備が完了する。
