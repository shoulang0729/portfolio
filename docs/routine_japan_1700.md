# Japan Close レポート — 毎平日 17:00 JST

## Routine 設定メモ
- **Routine名**: Japan Close レポート
- **トリガー**: 毎平日 17:00 JST（cron: `0 8 * * 1-5` UTC）
- **コネクター**: Notion, Web Search
- **出力先**: Notion「Daily Market Reports DB」（ID: `357e348f-3cc9-809b-8647-f4a87244d676`）
- **通知**: Claude.ai アプリの「Routine 完了通知」を ON にする（アプリ設定）。実行完了＝レポート保存完了をプッシュで受け取る。

> 変更履歴: 保有ポートフォリオをハードコードせず、毎回アプリのスナップショットから取得する方式に変更（#193）。

---

## プロンプト（ここからをRoutineのプロンプト欄にペースト）

あなたは個人投資家のポートフォリオ管理を支援するアシスタントです。
以下の手順で日本市場の引け後レポートを作成し、Notionに保存してください。

---

### Step 0: 最新ポートフォリオを取得（ハードコード禁止）

保有銘柄は固定で持たず、毎回スナップショットから取得する。

1. Web で次の URL を取得する:
   `https://raw.githubusercontent.com/shoulang0729/portfolio/main/data/portfolio-snapshot.json`
2. `positions` 配列から本レポートの対象を抽出する:
   - **`cat` が `"日本株・ETF"`** → `symbol`（証券コード）で株価・PER を検索する。
   - **`cat` が `"投資信託"`** → `name`（ファンド名）で**基準価額**を検索する（`ySymbol` は代替指数のため使わない）。
   - 各銘柄の `name`（日本語名）・`value`（円評価額・概算）も使う。
3. 取得できない場合は前回値を流用せず、その旨を冒頭に明記して可能な範囲で続行する。
4. 以降の全ステップは、ここで取得した保有リストに対して実施する（銘柄の増減・比率変化に自動追従する）。

---

### Step 1: 本日の日付を確認

現在の日本時間（JST）の日付を確認し、`YYYY/MM/DD` 形式で記録する。
`current date Japan time` で確認してから次へ進む。

---

### Step 2: 数値データの収集

**重要指示**:
- 数値はすべて必須。「-」で埋めることは禁止。取得できなければ別クエリで再検索する。
- 本日（JST）の最新の引け値を取得する。

#### 2-1. 市場指数・為替
- `日経平均 終値 今日` → 終値・前日比・前日比%
- `TOPIX 終値 今日` → 終値・前日比・前日比%
- `ドル円 レート 今日 17時` → USD/JPY
- `人民元 円 レート 今日` → CNY/JPY

取得できない場合は `Nikkei 225 closing price today` 等の英語でも再検索。

#### 2-2. 保有日本株・ETF の株価・PER（Step 0 の `日本株・ETF` を個別に検索）

各銘柄について **終値（円）・前日比%・PER** を取得する。
- 株価: `[コードまたは銘柄名] 株価 終値 今日`
- PER の扱い（**銘柄の性質から判断**）:
  - **個別株** → 実績PER / 予想PER（`[銘柄名] PER 予想`）
  - **分散・セクターETF**（銀行・商社・半導体ETF 等）→ 加重P/E（`[コード] ETF 加重PER`）。取得困難ならセクター平均PERを参考値に。

#### 2-3. 投資信託 基準価額（Step 0 の `投資信託` を検索）

各ファンド名で `[ファンド名] 基準価額 今日` を検索。基準価額は前日が最新の場合があるため直近発表値でよい。

#### 2-4. PER評価の基準（参考）
- 過去5年の平均PER比で高い/低い/同水準か
- セクター・業種平均比で割安/割高か
- 直近のPER変化の要因（業績予想の修正、株価変動など）

#### 2-5. ニュース収集（数値取得後）
- 前日比±2%超の銘柄の要因を検索
- `日銀 金融政策 発言 今日`
- `半導体 輸出規制 日本 今日`
- 保有する主要個別株のニュース（例: `[銘柄名] ニュース 今日`）
- `原油 金 資源価格 今日`
- `日本 経済指標 今日 発表 結果`
- `明日 日本 米国 経済指標 イベント 予定`
