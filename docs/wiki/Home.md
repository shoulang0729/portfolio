# Portfolio Manager — Wiki

**Portfolio Manager** は Finnhub / Yahoo Finance API を使ったポートフォリオ可視化 PWA です。  
GitHub Pages でホストされ、PIN / パスキー認証でプライベートに管理できます。

- **本番 URL**: https://shoulang0729.github.io/portfolio/
- **リポジトリ**: https://github.com/shoulang0729/portfolio

---

## 主な機能

| タブ | 内容 |
|---|---|
| **Heatmap** | 保有銘柄の損益をヒートマップで可視化。セルをタップするとチャートが開く |
| **Historical Heatmap** | 1d / 5d / 1m / 6m / 1y / 5y / 10y の期間別騰落率テーブル |
| **Watchlist Historical Heatmap** | ウォッチリスト銘柄の同じ期間別テーブル。ウォッチリストは KV でクロスデバイス同期 |
