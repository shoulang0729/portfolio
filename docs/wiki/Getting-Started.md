# はじめに（Getting Started）

## アクセス

https://shoulang0729.github.io/portfolio/ をブラウザで開いてください。  
iOS Safari / Android Chrome / PC Chrome に対応しています。  
ホーム画面に追加すると PWA として動作します（オフライン閲覧も可）。

---

## ログイン

初回アクセス時に PIN（4桁）の入力画面が表示されます。

| 方法 | 手順 |
|---|---|
| PIN | 4桁の数字を入力 |
| パスキー | ログイン後、メニュー → 「パスキー登録」で Touch ID / Face ID を登録できます |

> PIN を変更する場合: メニュー → 「PIN変更」

---

## 保有銘柄の更新

保有銘柄は3通りの方法で更新できます。

### 1. マネックス証券 CSV 取込
1. マネックス証券 → 保有商品一覧 → CSV ダウンロード
2. アプリ メニュー → 「マネックス取込」→ CSV ファイルを選択

### 2. マネーフォワード スクリーンショット取込
1. マネーフォワード ME のポートフォリオ画面をスクリーンショット
2. アプリ メニュー → 「マネフォ取込」→ 画像を選択

### 3. ソースコード直接編集（開発者向け）
`src/positions.js` の `positions` 配列を編集してコミット・プッシュ。

---

## ウォッチリスト

- Heatmap タブ右上の「検索」から銘柄を追加
- 追加した銘柄は Watchlist Historical Heatmap タブに表示される
- Cloudflare KV に自動同期されるためスマホ・PC 間で共有

---

## スナップショット保存

ログイン後、メニュー → 「スナップショット保存」で現在のポートフォリオを  
`data/portfolio-snapshot.json` に保存できます（他の AI ツール連携用）。

---

## 自動更新

メニュー → 自動更新 で OFF / 5分 / 10分 / 30分 / 60分 を選択できます。
